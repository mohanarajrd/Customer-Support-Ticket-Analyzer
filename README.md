
# 🎫 Customer Support Ticket Analyzer

A Python-based project that simulates a customer support ticket management system. The application allows users to add support tickets, clean issue descriptions, analyze recurring keywords, categorize ticket priorities, and generate summary reports to provide meaningful operational insights.

---

## 📌 Project Overview

Customer support teams receive numerous service requests every day. Efficient analysis of these tickets helps identify recurring customer issues, prioritize critical requests, and improve overall support quality.

This project demonstrates how core Python programming concepts can be applied to build a simple ticket analysis system capable of processing customer complaints and generating useful business insights.

---

## 🎯 Objectives

- Manage customer support tickets
- Validate user inputs
- Clean and preprocess ticket descriptions
- Perform keyword-based issue analysis
- Categorize tickets by priority
- Generate summary reports and insights

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Built-in Python Libraries

---

## 📂 Dataset

The project uses a predefined collection of customer support tickets containing:

- Ticket Number
- Customer Name
- Issue Description
- Priority Level

Users can also dynamically add new support tickets during program execution.

---

## ⚙️ Features

### Ticket Management

- Preloaded customer support tickets
- Add new tickets dynamically
- Automatic ticket number generation

### Input Validation

- Validates priority values
- Prevents invalid user entries

### Text Preprocessing

- Converts text to lowercase
- Removes punctuation
- Cleans unnecessary characters
- Standardizes issue descriptions

### Keyword Analysis

Searches ticket descriptions for common keywords such as:

- Poor
- Good
- Slow
- Internet
- Support

### Priority Analysis

Categorizes tickets into:

- 🔴 High Priority
- 🟡 Medium Priority
- 🟢 Low Priority

### Reporting

Generates:

- Total number of tickets
- Priority distribution
- Keyword frequency analysis
- Final cleaned ticket dataset
